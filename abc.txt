Java的优缺点：
1.与C++想比，尽管他的运行时库是用C语言来开发的，但是他不依赖C，而且他开发一次就能一直运行（跨平台）；
2.与C#相比，Java适合于所有的系统，但是C#有局限性只能被微软自己的系统使用。

JVM:java的虚拟机，仿真模拟真正的计算机的功能来进行的。他屏蔽了与具体的OS的联系，所以程序只需要在虚拟机上运行的目标代码，就可以在多种平台上运行。
JDK：java开发工具包，是整个的核心
JRE：java运行环境，也可以直接说是所有环境的集合。
JAVA SE：标准版，用于桌面软件的编程。
JAVA EE：企业版，用于分布式网络程序的开发。
JAVA ME：是提供Java语言的一个平台。
GC：垃圾回收，虚拟机会自动清理没有用的对象。
